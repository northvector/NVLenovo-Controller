# NVLenovo-Controller
###### Edited version of [Lenovo controller](https://github.com/ViRb3/LenovoController)

### Tested on lenovo ideapad gaming 3 

### New features 
- Better UI 
- Power plan changer
- System tray GPU + CPU temp monitor
- WIFI Fix
- Brightness control
- Timed shutdown
- DNS Changer
